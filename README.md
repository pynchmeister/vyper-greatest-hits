# vyper-greatest-hits ⚡️

A curated collection of smart contracts and examples written in Vyper — intended to illustrate Vyper concepts, patterns, and best practices.  

---

## ✅ Why This Repository

- Provide a hands-on set of Vyper contracts covering a variety of use cases, from simple utilities to more advanced patterns.  
- Serve as reference code for developers — whether you're learning Vyper or building real projects.  
- Demonstrate how to structure, test, and maintain Vyper projects using standard tooling (e.g. scripting + testing frameworks).
